A simple plugin for converting images in webp and render webp formate images in frontend.

== Description ==

A simple plugin for converting images to webp (when uploading) and this plugin also render webp images to frontend.

A free, lightweight plugin and easy to use plugin for converting images to webp is ready for you. 

For basic usage, it just need to install and activate in your Wordpress.

== Installation ==

1.Upload the entire `convert-images-in-webp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

OR

1. **Visit** Plugins > Add New
2. **Search** for "Convert Images In Webp"
3. **Install and Activate** Convert Images In Webp from your Plugins page
